# caretaker_app (Flutter — iOS + web)

Plain Flutter project (intentionally **not** Nx-managed; driven by the `flutter`
CLI). The Flutter SDK is required and is not installed by the backend toolchain.

## First-time setup

The platform runner directories (`ios/`, `web/`) are generated by Flutter, not
committed here. After installing Flutter:

```bash
cd apps/mobile
flutter create . --platforms=ios,web --project-name caretaker_app
flutter pub get
```

This fills in `ios/` and `web/` without overwriting `lib/main.dart` or
`pubspec.yaml`.

## Run

```bash
flutter run -d chrome      # web
flutter run -d ios         # iOS simulator
```

## Test / analyze

```bash
flutter analyze
flutter test
```

## API client

The typed Dart API client is generated from the backend's OpenAPI spec into
`lib/api/generated/` (see `/tools`). It is git-ignored and regenerated.
